Current

Fixed height + Contain

The image element has a fixed height: 220px and object-fit: contain. The image is scaled to fit inside the box — correct aspect ratio, but the remaining space becomes empty letterbox bands. Drag the panel wider: the bands grow, the image stays the same size.

Equivalent aetna code:

image(img).width(Fill(1.0)).height(Fixed(220.0)).image_fit(Contain)

Desired

Intrinsic height (width: 100%, height: auto)

The image element fills the available width and derives its height from the image's natural aspect ratio — no letterbox bands. Drag the panel wider: the image grows proportionally. This is CSS width: 100%; height: auto on an <img>.

Desired aetna code:

image(img).width(Fill(1.0)).height(Hug) (with aspect-ratio-aware Hug)
